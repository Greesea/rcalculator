# Code计算器

自带的计算稿纸不太能满足我的需求，工作需要经常需要计算各种成本，算式结果要复用。
刚好在学编译原理，就简单粗暴写了个计算器。

变量： a = -1.2  
默认变量：E、LN2、LN10、LOG2E、LOG10E、PI、SQRT1_2SQRT2  
可以重新赋值：E = 0  
运输符：+、-、*、/、%、()  
数学函数：abs(-1.2)、max(0.5, -2.1)、random() 等，参考【[Math](https://www.w3school.com.cn/jsref/jsref_obj_math.asp)】
